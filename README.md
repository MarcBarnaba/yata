# YATA — Yet Another TODO App

A GTD (Getting Things Done) productivity app built with Nuxt 3, Vue 3, Pinia, and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- **Nuxt 3** — Vue meta-framework
- **Vue 3** — Composition API
- **Pinia** — State management
- **Tailwind CSS** — Utility-first styling
- **TypeScript**
